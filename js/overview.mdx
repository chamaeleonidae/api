## Key Methods

The JavaScript API provides several key methods:

### Identify Users
```javascript
chmln.identify(userId, {
  email: 'user@example.com',
  name: 'John Doe'
});
```

### Track Events
```javascript
chmln.track('Button Clicked', {
  button_name: 'signup',
  page: 'homepage'
});
```

### Show Experiences
```javascript
chmln.show('tour-id-123');
```
