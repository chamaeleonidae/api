Experience Deliveries allow you to trigger Tours, Microsurveys, and other Chameleon Experiences via API.

## Schema

| Property | Type | Description |
| --- | --- | --- |
| `id` | ID | The Chameleon ID |
| `kind` | string | The type of Experience: `tour`, `survey`, `launcher` |
| `experience_id` | ID | The ID of the Experience to deliver |
| `profile_id` | ID | The Chameleon Profile ID to deliver to |
| `created_at` | timestamp | When this delivery was created |
| `delivered_at` | timestamp | When this was actually delivered |

## Create a Delivery

### HTTP Request

```http
POST https://api.chameleon.io/v3/edit/deliveries
```

### Request Body

```json
{
  "kind": "tour",
  "experience_id": "5f3c4232c712de665632a6d5",
  "profile_id": "5f3c4232c712de665632a6d6"
}
```

## List Deliveries

### HTTP Request

```http
GET https://api.chameleon.io/v3/edit/deliveries
```

| param | - | description |
| ------ | -------- | ------------------------------------------------------------ |
| `limit` | optional | Defaults to `50` with a maximum of `500` |
| `before` | optional | Used when paginating |
| `after` | optional | Read as "created `after`" timestamp or ID |
