title: "Properties"
---

Properties track the specific data sent to Chameleon via custom data properties to [User Profiles](apis/profiles.md) or [Companies](apis/companies.md).

## Schema

<ResponseField name="id" type="ID">
  The Chameleon ID
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  When this happened or when this was added to the Database
</ResponseField>

<ResponseField name="updated_at" type="timestamp">
  The last time any property was updated
</ResponseField>

<ResponseField name="name" type="string">
  The name given by an administrator of Chameleon
</ResponseField>

<ResponseField name="description" type="string">
  The display description
</ResponseField>

<ResponseField name="kind" type="string">
  Type of record this property is associated with: One of `profile`, `event`, or `company`
</ResponseField>

<ResponseField name="prop" type="string">
  The normalized property key
</ResponseField>

<ResponseField name="integration" type="string">
  The source integration of this property
</ResponseField>

<ResponseField name="last_seen_at" type="timestamp">
  The time when this property was last added to / removed from a User Profile.
</ResponseField>

<ResponseField name="types" type="array">
  The identified type of property values seen (string, integer etc.)
</ResponseField>

<ResponseField name="values" type="array">
  A sample of the most recent values seen for this property (most recent first).
</ResponseField>

## Listing Properties

Retrieve a complete (un-paginated) list of properties for the given `kind` + `integration`.

#### HTTP Request

```
GET https://api.chameleon.io/v3/edit/properties
```

<ParamField query="kind" type="required">
  Use a values of either `profile` or `company`
</ParamField>

<ParamField query="integration" type="optional">
  Used to filter properties to only ones created by a specific integration. Omit for "all integrations".
</ParamField>

<ParamField query="order" type="optional">
  One of `created_at` or `last_seen_at`, ordered by most recent. Defaults to `created_at`.
</ParamField>

Custom User Profile properties
```json
{
  "kind": "profile",
  "integration": "api",
  "order": "last_seen_at"
}
```

Salesforce Company properties
```json
{
  "kind": "company",
  "integration": "salesforce"
}
```

#### HTTP Response

```json
{
  "properties": [
    {
      "id": "5f3c4232c712de665632a6d9",
      "name": "Monthly plan value (dollars USD)",
      "prop": "plan_cost",
      "types": ["integer"],
      "values": [678, 442, 1218, 239, 394],
      "last_seen_at": "2029-04-07T12:18:00Z"
    },
    {
      "id": "5f3c4232c712de665632a6e2",
      "name": "Lifetime value",
      ...
    },
    ...
  ]
}
```

## Create a Property

#### HTTP Request

```
POST https://api.chameleon.io/v3/edit/properties
```

<ParamField body="name" type="string" required>
  The name given by an administrator of Chameleon
</ParamField>

<ParamField body="description" type="string" required>
  The display description
</ParamField>

<ParamField body="kind" type="string" required>
  Type of record this property is associated with: One of `profile`, `event`, or `company`
</ParamField>

```json
{
  "name": "Next invoice date",
  "description": "The next time an Invoice will be generated by QBO",
  "kind": "company",
  ...
}
```

#### HTTP Response

```json
{
  "property": {
    "id": "5f3c4232c712de665632a6d5",
    "created_at": "2029-02-03T00:00:00Z",
    "name": "Next invoice date",
    "description": "The next time an Invoice will be generated by QBO",
    "kind": "company",
    "prop": "next_invoice_date",
    ...
  }
}
```
