## Installing Chameleon

To install Chameleon on your website, add the following script tag to your HTML:

```html
<script src="https://fast.chameleon.io/mutable/YOUR_ACCOUNT_ID.js"></script>
```

## Verification

After installation, you can verify Chameleon is working by:

1. Opening your browser's developer console
2. Typing `chmln` - you should see the Chameleon object
3. Check for any console errors
